# 🍽️ Cafe Menu — Электронное меню для кафе

Мобильное веб-приложение для просмотра меню и заказа блюд в кафе.

## 📋 Описание

Проект представляет собой систему электронного меню с возможностью оформления заказов. Приложение состоит из клиентской части (для посетителей) и админ-панели (для управления меню и просмотра заказов).

## ✨ Возможности

### Для посетителей:
- Просмотр категорий блюд
- Просмотр меню с фотографиями
- Добавление блюд в корзину
- Оформление заказа

### Для администратора:
- Авторизация в системе
- Управление категориями блюд
- Управление блюдами (добавление, редактирование, удаление)
- Загрузка изображений блюд
- Просмотр всех заказов

## 🛠️ Технологии

**Frontend:**
- React 18
- React Router DOM
- Vite

**Backend:**
- Node.js
- Express.js
- SQLite3
- JSON Web Token (JWT)
- Multer (загрузка файлов)

## 🚀 Быстрый старт

### Требования
- Node.js >= 16
- npm

### Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd cafe-menu
```

2. Установите все зависимости:
```bash
npm run install:all
```

### Запуск в режиме разработки

Запуск одновременно сервера и клиента:
```bash
npm run dev
```

Или по отдельности:

**Только сервер:**
```bash
npm run dev:server
```

**Только клиент:**
```bash
npm run dev:client
```

Приложения будут доступны по адресам:
- Клиент: `http://localhost:5173`
- Сервер: `http://localhost:3001`

## 🔐 Настройка

Создайте файл `.env` в корне проекта для настройки переменных окружения:

```env
JWT_SECRET=your_secret_key_here
PORT=3001
```

## 📡 API Endpoints

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/categories` | Получить все категории |
| POST | `/api/categories` | Создать категорию |
| GET | `/api/dishes` | Получить все блюда |
| POST | `/api/dishes` | Создать блюдо |
| POST | `/api/orders` | Оформить заказ |
| GET | `/api/admin/orders` | Получить все заказы (требуется авторизация) |
| POST | `/api/auth/login` | Войти как администратор |

## 📱 Использование

### Первый запуск

1. Запустите приложение командой `npm run dev`
2. Откройте браузер по адресу `http://localhost:5173`
3. Для доступа к админ-панели перейдите на `/admin` и выполните вход

### Админ-панель

В админ-панели вы можете:
- Добавлять новые категории блюд
- Создавать блюда с фотографиями
- Просматривать поступившие заказы
